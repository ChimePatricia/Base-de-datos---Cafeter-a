# Base-de-datos---Cafeter-a
El propósito de este proyecto es generar una base de datos que ayude a la perfecta administración de una cafetería, en donde no solo se atenderán a los clientes con bebidas o aperitivos, sino que tambien tendrán la oportunidad de convivir con gatos rescatados de la calle en el cual se  sientan seguros y se les busque un hogar.
